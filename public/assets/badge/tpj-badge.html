TPJ認証バッジ 設置ガイド

Third Place Japan（サードプレイスジャパン）認証を取得した施設のウェブサイトに設置いただけるバッジコンポーネントです。

バッジのプレビュー

TPJ
Certified by Third Place Japan Flagship
TPJ
Certified by Third Place Japan Platinum
TPJ
Certified by Third Place Japan Gold
TPJ
Certified by Third Place Japan Silver
TPJ
Certified by Third Place Japan Certified

左から：Flagship（標準）/ Platinum（標準）/ Gold（標準）/ Silver（スモール）/ Certified（スモール）

設置手順

以下のコードをウェブサイトの <head> 内にCSSを、バッジを表示したい場所にHTMLを貼り付けてください。

Step 1：CSSをコピーして <head> 内に貼り付ける

<style>
.tpj-badge {
  display: inline-flex;
  align-items: center;
  gap: 10px;
  padding: 10px 16px;
  border: 1px solid #B8960C;
  background: #fff;
  text-decoration: none;
  font-family: 'Helvetica Neue', Arial, sans-serif;
  transition: background 0.2s, box-shadow 0.2s;
}
.tpj-badge:hover { background: #FDFBF5; box-shadow: 0 2px 8px rgba(184,150,12,0.15); }
.tpj-badge__logo-mark {
  width: 32px;
  height: 32px;
  border: 1.5px solid #B8960C;
  display: flex;
  align-items: center;
  justify-content: center;
  font-size: 11px;
  font-weight: 600;
  letter-spacing: 0.05em;
  color: #B8960C;
}
.tpj-badge__body { line-height: 1.3; }
.tpj-badge__label { display: block; font-size: 9px; letter-spacing: 0.12em; color: #888; text-transform: uppercase; }
.tpj-badge__name  { display: block; font-size: 13px; font-weight: 500; color: #1C1C1C; }
.tpj-badge__grade { display: inline-block; padding: 2px 8px; font-size: 10px; font-weight: 600; letter-spacing: 0.08em; color: #fff; margin-top: 2px; }
.tpj-badge__grade--flagship { background: #1C1C1C; }
.tpj-badge__grade--platinum { background: #5C5C7A; }
.tpj-badge__grade--gold     { background: #B8960C; }
.tpj-badge__grade--silver   { background: #888; }
.tpj-badge__grade--certified { background: #4A7A4A; }
</style>

Step 2：バッジHTMLをページに貼り付ける（グレードを変更して使用）

認証グレードに合わせて tpj-badge__grade--{グレード} クラスと表示テキストを変更してください。

<!-- Flagship の場合 -->
<a href="https://thirdplacejapan.com/stories/certified/"
   target="_blank" rel="noopener" class="tpj-badge">
  <div class="tpj-badge__logo">
    <div class="tpj-badge__logo-mark">TPJ</div>
  </div>
  <div class="tpj-badge__body">
    <span class="tpj-badge__label">Certified by</span>
    <span class="tpj-badge__name">Third Place Japan</span>
    <span class="tpj-badge__grade tpj-badge__grade--flagship">Flagship</span>
  </div>
</a>

<!-- Gold の場合 -->
<a href="https://thirdplacejapan.com/stories/certified/"
   target="_blank" rel="noopener" class="tpj-badge">
  <div class="tpj-badge__logo">
    <div class="tpj-badge__logo-mark">TPJ</div>
  </div>
  <div class="tpj-badge__body">
    <span class="tpj-badge__label">Certified by</span>
    <span class="tpj-badge__name">Third Place Japan</span>
    <span class="tpj-badge__grade tpj-badge__grade--gold">Gold</span>
  </div>
</a>

グレード対応表

クラス名とグレード表示テキストの対応は以下のとおりです。

Flagship  → class="tpj-badge__grade tpj-badge__grade--flagship"  ／ テキスト: Flagship
Platinum  → class="tpj-badge__grade tpj-badge__grade--platinum"  ／ テキスト: Platinum
Gold      → class="tpj-badge__grade tpj-badge__grade--gold"      ／ テキスト: Gold
Silver    → class="tpj-badge__grade tpj-badge__grade--silver"    ／ テキスト: Silver
Certified → class="tpj-badge__grade tpj-badge__grade--certified" ／ テキスト: Certified

注意事項

・バッジのリンク先（href）は必ず https://thirdplacejapan.com/stories/certified/ にしてください。

・バッジ画像の改変、グレードの改ざんは禁止しています。

・認証が失効した場合は速やかにバッジを削除してください。

・ご不明な点は thirdplacejapan.com/contact までお問い合わせください。
