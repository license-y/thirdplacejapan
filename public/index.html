Third Place Japan

どこにいるか、を定義する。

丁寧に場をつくる人たちのための、日本初のサードプレイス認証。

TPJ — 認証の中立性

「認証は、買えない。」

評価・グレードは覆面調査員による独立した判断に基づきます。いかなるご支援のご利用も、認証の可否やグレードに影響を与えることはありません。

Philosophy

効率の先にあるもの

かつて、場所を選ぶ基準は「効率」だった。近くて、安くて、便利であること。

けれど、時代は静かに変わり始めている。人々が本当に求めているのは、「早く済ませられる場所」ではなく、「そこにいたいと思える場所」だ。

居心地。静けさ。自分の時間。日常から少し離れた、特別な体験。そうした価値は、これまで正しく言葉にされてこなかった。口コミの星の数や、検索順位では測れない。

Third Place Japanは、この「いる体験の価値」を見つめ直すための小さな視点として生まれた。

Overview

掲載の全体像

TPJには2つの掲載形態があります。編集部が申請不要で発掘・紹介する「TPJセレクト」と、施設様からのお申し込みによる「正式認証（Certified以上）」です。TPJセレクトが裾野を広く持つ入口となり、そこから正式認証への経路が続きます。

Case Study — Flagship

言葉だけでなく、実践として

Photo by Yanapi Senaud / Unsplash

TPJが見つめてきた価値観は、机上のものではない。

GBC（Green Beans Coffee）は、その価値観を実際の店舗で試し、検証し続けている場所である。

席の間隔、光の当て方、音の設計、声をかけるタイミング——ひとつひとつを7基準に照らし、積み重ねた知見を評価基準そのものに還元している。

理念は、そこに「在る」ことで初めて、伝わるものになる。

FAQ

よくある質問

Certified認証の取得に費用はかかりますか？
Certified認証に費用は一切かかりません。自薦・他薦を問わず申請でき、簡易的な情報の登録のみで審査を受けられます。Silver以上をご検討の場合は、詳細をお問い合わせください。
Silver以上の認証はどのように申請しますか？
Silver以上の認証は、施設オーナーまたは責任者本人からのお問い合わせが必要です。第三者が代理申請することはできません。お問い合わせ後、TPJ編集部との初回ヒアリングを経て審査が始まります。
掲載枠や記事制作、空間プロデュースを利用しないと認証されませんか？
いいえ。掲載枠・記事制作・空間プロデュース・プロダクト化等のご支援のご利用は、認証の可否やグレードに一切影響しません。これらは認証とは独立した任意のご支援です。評価は覆面調査員の独立した判断に基づきます。
認証までどのくらいの期間がかかりますか？
グレードによって異なります。Certifiedは基礎確認のみで比較的短期間です。Silver以上は複数回の覆面調査と合議を経るため、数週間から数ヶ月かかる場合があります。詳細はお問い合わせください。
掲載枠にはどのような種類がありますか？
地域×業種で数に限りのある特別な掲載枠をご用意しています。ご希望の形態や空き状況については、For BusinessページよりTPJへお問い合わせください。
認証後、グレードは変動しますか？
はい。TPJは継続的な評価を行います。空間の質が維持・向上していれば上位グレードへの昇格があります。また、認証基準を満たさなくなった場合には、グレードの見直しが行われることもあります。これは認証の誠実さを保つための仕組みです。

評価プロセスの独立性について

TPJの認証は、覆面調査員が独立した基準に基づいて行います。調査員の身元・訪問日は事前に施設へ通知されません。評価スコアと認証グレードの決定プロセスに、掲載枠・記事制作・空間プロデュース等のご支援は一切関与しません。Certifiedグレードは費用が一切かかりません。Silver以上のグレード審査も、オーナーとの対話と調査員の記録に基づくものであり、対価によって得られるものではありません。この独立性こそ、TPJ認証の信頼の根拠です。

効率の先にあるもの

TPJは2026年に創設されました。食べログが「飲食店の評価」を定義したように、TPJは「いる体験の価値」を定義する存在を目指しています。7つの評価軸は、社会学者レイ・オルデンバーグのサードプレイス理論を現代日本に再解釈したものです。居心地・静寂性・特別感・ストーリー・再訪価値・記録体験・インバウンド対応という7軸は、口コミでも星の数でも捉えきれなかった「いる体験」を構造化します。

見つける — Discover

TPJの調査員は、広告費をかけていない場所、目立たない場所にこそ注目します。規模の大きさではなく、その場所が大切にしてきた在り方——光の当て方、音の設計、訪れた人への敬意——を探しています。
お問い合わせはこちら →

見つめる — Evaluate

7つの評価軸すべてに対し、複数回の訪問記録をつけます。調査員の主観ではなく、具体的な観察事実と基準への照合に基づきます。Silver以上では、複数の調査員による合議を経ます。
お問い合わせはこちら →

認める — Certify

認証は、その場所の在り方への承認です。競争のための順位ではなく、その場所が積み重ねてきたものを言葉にして残すために行います。Certifiedは誰でも入口に立てる公平な第一層であり、Silver以上はオーナー本人との対話を経た、より深い認証です。
お問い合わせはこちら →

伝える — Publish

認証内容は日本語と英語の両方で発信します。構造化データ（JSON-LD）により、人だけでなくAIにも正確に届く設計です。国内外のメディア・AI検索エンジンへの引用を通じて、その場所の価値が世界に伝わります。
お問い合わせはこちら →

居心地・空間品質 → 五感品質・素材

空間の居心地を生む要素——光、音、温度、素材の選択——を、プロダクトの素材・香り・パッケージへ翻訳します。認証された空間のアンビエンスを手元で再現できるかを評価します。
お問い合わせはこちら →

静寂性・プライバシー → 静けさの再現

その場所が持つ静けさ——音設計、視線の配慮、空間の余白——を、プロダクトがどこまで日常に持ち込めるかを見ます。使用する時間に「自分だけの時間」が生まれるか。
お問い合わせはこちら →

特別感・非日常性 → 特別感・ギフト体験

その空間でしか得られない体験——建築、眺望、季節の演出——を、プロダクトの開封体験・ギフト包装・限定性へ翻訳します。受け取った人が「特別な何か」を感じるかを評価します。
お問い合わせはこちら →

ストーリー・背景への共感 → ストーリー継承

その場所が持つ物語——創業の経緯、職人のこだわり、地域との関係——を、プロダクトのコピー・ブックレット・QRコンテンツで伝えられるかを見ます。プロダクトそのものが語り部になっているか。
お問い合わせはこちら →

再訪・継続価値 → 愛用・回遊価値

空間への再訪を促す仕掛け——季節限定メニュー、常連への特別な配慮——が、プロダクトではリピート利用・シリーズ展開・リフィル設計などに翻訳されます。一回で終わらない関係性を設計できているか。
お問い合わせはこちら →

記録・シェア体験 → シェア・記録性

空間での「誰かに見せたい」衝動——映える角度、特別な照明、記念になる瞬間——が、プロダクトでは視覚設計・SNS映えするパッケージ・ハッシュタグ設計へと翻訳されます。
お問い合わせはこちら →

インバウンド・多言語対応 → インバウンド適性

空間での英語メニュー・スタッフ対応・文化的配慮が、プロダクトでは多言語パッケージ・国際配送対応・海外向けギフト展開に翻訳されます。「東京みやげ」として世界に渡せるか。
お問い合わせはこちら →

Certified — 開かれた入口

Certifiedは、誰の元にも開かれた第一層です。自薦・他薦を問わず、簡易的な情報があれば、まずその場所を「第三の居場所」として記録します。費用は一切かかりません。表示は基本情報と7基準の概要のみのシンプルな形ですが、「言葉にされ、記録される」ことそのものが意味を持ちます。
お問い合わせはこちら →

Silver — 対話の始まり

Silverから、オーナーや施設責任者本人との対話が始まります。問い合わせの後、TPJ編集部とのヒアリングを経て、覆面調査員が複数回訪問します。7基準スコアが明示され、その場所のストーリー・哲学をページに掲載することが可能になります。「静かに、丁寧に運営されている場所」としての認証です。
お問い合わせはこちら →

Gold — 継続の証明

Silver認証を経た施設が、継続的な調査・対話を経て到達するグレードです。一度の評価ではなく、時間をかけた観察によって「質が持続している」ことが確認された場所です。詳細な認証ページ（ストーリー・写真を含む）が掲載され、メディアでの紹介機会が増えます。
お問い合わせはこちら →

Platinum — 稀有である

「質が高い」を超え、「稀有である」と認められたグレードです。複数チームによる複数回の再評価を経て、7基準すべてにおいて高水準であることが確認されます。国内外への発信（英語版含む）で優先的に取り上げられます。
お問い合わせはこちら →

Flagship — 理念の体現

TPJが「理想的なサードプレイスの理念を最も誠実に体現している」と認めた最上位の認証です。編集部による最終審査（覆面調査記録の総合評価＋編集責任者の確認）を経ます。「最も高いものを売る」場所ではなく、TPJが大切にしてきた世界観を、最も誠実に表している場所です。
お問い合わせはこちら →

サードプレイスの空間プロデュース

TPJの7基準（居心地・静寂性・特別感・ストーリー・再訪価値・記録性・インバウンド適性）に基づき、空間の設計・改善をご支援します。既存のレイアウト、音環境、照明、素材選択を7基準の観点から評価し、具体的な改善案を提案します。詳細はお問い合わせください。
お問い合わせはこちら →

7つの基準のプロダクト化

空間で体現された7基準を、持ち帰れるプロダクト（ギフト・グッズ等）へ翻訳する企画・監修をご支援します。上記の「7基準対応表」で示した翻訳の考え方に基づき、プロダクトのコンセプト立案から素材選定・ブランディングまでをサポートします。詳細はお問い合わせください。
お問い合わせはこちら →

Green Beans Coffee — TPJ Flagship認証

GBC（Green Beans Coffee）は、TPJ初のFlagship認証施設であり、TPJが提唱する「いる体験」の設計思想を実店舗で検証し続けている実証店舗です。席の間隔、照明の明るさ、音の設計、接客の姿勢——空間づくりの一つひとつを7基準に照らして継続的に見直し、そこで得られた知見をTPJの評価基準や審査の判断材料として反映しています。
