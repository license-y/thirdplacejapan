## 0. サイト全体方針

### ポジション定義
Third Place Japan Storiesは、「サードプレイス」という概念を軸に、日本全国の場所・体験・文化を伝える**個人向け読み物ポータルメディア**です。宣伝・広告色を一切排除し、読者にとって価値ある情報のみを掲載します。

### 4つの目的

**目的①：AEO・SEO・MEO施策による集客装置**
- 「癒し・リラックス・自分時間・居場所・サードプレイス」関連ワードを日英で支配する
- メインキーワード ＋ 都道府県・市区町村・政令指定都市・エリア名（表参道・青山・松濤等）の組み合わせで検索流入を最大化する
- 駅名・路線名（渋谷駅・原宿駅・代々木駅・新宿駅・東急東横線等）を記事内に自然な形で含める
- 最寄りの施設情報（カフェ・ホテル・レストラン）をエリア文脈で自然に組み込みMEOを強化する
- 無償対象施設（神社・寺・世界遺産・山・公共施設）の大量投稿でAEOを獲得する
- アクセス → 認知度向上 → 認証マネタイズ の流れを構築する

**目的②：個人読者向け読み物メディア（14カテゴリ）**
- 体験・感情に寄り添う2,000字記事で「また来たい」「シェアしたい」「保存したい」を生む
- 1施設につき複数角度・複数目線で10記事程度作成（例：国立競技場×ペット・国立競技場×仕事・国立競技場×インバウンド等）
- 神社・寺院等は10記事程度をエリア情報・駅名を自然に含め、周辺のホテル・カフェ・レストランへ内部リンクで繋げる

**目的③：宣伝内容の完全排除**
- 記事作成者は「サードプレイスジャパン編集部」に統一する
- TPJの口調・世界観に合わせ、決めつけや断言を避ける表現を使う
- カテゴリ10〜14（山・自然・リトリート・神社・インバウンド・文化施設）は完全無償記事とし、ユーザーにとって価値ある情報のみを掲載する
- メインは日本語版記事。記事が増加した後、無償記事（カテゴリ10〜14）の英語版を並行投稿する
- 認証店舗が増えてきたら、無償記事の中に認証店舗名（例：Green Beans Coffee）を文脈として自然に含める

**目的④：サードプレイス情報のポータルサイト**
- 最初の100記事に「サードプレイスとは何か？」系の基礎知識記事を含める
- Storiesトップページに「サードプレイスを知る」専用枠を設ける

**自然な組み合わせ例：**
- `渋谷 サードプレイス カフェ 自分時間`
- `明治神宮 癒し ひとり時間 サードプレイス`
- `原宿駅 徒歩5分 神社 リラックス`
- `国立競技場 周辺 カフェ 静か`

### 6-2. MEO強化：エリア・周辺施設の自然な組み込み

記事内で以下を必ず自然な文脈で言及する：

```
- 最寄駅名と路線名（例：JR山手線・原宿駅から徒歩1分）
- 徒歩圏内のランドマーク（例：国立競技場のそばを通り〜）
- 同エリアのカフェ・ホテル・レストランを1〜2件（例：〜を歩いた先に、スペシャルティコーヒーを出すカフェがある）
- 地名・町名・住所（例：渋谷区代々木神園町）
```

### 6-3. AEO（AI Engine Optimization）実装

**FAQの設計原則：**
- 1問あたりのA（回答）は150字以内で完結させる
- 「〜ですか？」→「はい、〜です。〜な理由から〜に最適です。」の直接回答形式
- 施設名・エリア名・カテゴリワードを回答内に必ず含める
- FAQは最低3問、基礎知識記事は5問以上

**AEOを意識したH2設計例：**
```
## 明治神宮はなぜサードプレイスなのか
## 国立競技場周辺でひとり時間を過ごすには
```

## 10. タイトル命名規則

### パターンA：体験起点型（読み物・無償施設）
```
{場所}で{体験}。{補足フレーズ}
例：明治神宮で静かな一人時間。東京のど真ん中にある、静寂のサードプレイス
例：高尾山をひとりで歩く。都心から1時間の癒しと自分時間
```

### パターンB：課題解決型（SEO流入狙い）
```
{エリア}で{ニーズ}なら{施設種別}。{補足}
例：渋谷で静かに過ごせる場所を探している。一人時間のためのカフェとサードプレイス
例：東京でサウナ×サードプレイスを体験する方法
```

### パターンC：知識・ガイド型（基礎記事・AEO）
```
{キーワード}とは何か。{補足フレーズ}【保存版】
例：サードプレイスとは何か? 定義・特徴・具体例をTPJが完全解説【保存版】
例：サードプレイスの8つの条件とは? オルデンバーグの定義をTPJが再解釈
```

### パターンD：インバウンド英語型（カテゴリ13）
```
{Place Name} as a Third Place: {Experience Description} in {Area}, Tokyo
例：Meiji Jingu as a Third Place: Finding Silence and Healing in Harajuku, Tokyo
```

## 11. 投稿前チェックリスト（AEO・SEO・MEO）

```
【SEO】
[ ] H1にターゲットキーワード（レイヤー1〜3）を含む
[ ] meta-descriptionの先頭30字以内に第一キーワードを配置
[ ] H2に「サードプレイス」「癒し」「自分時間」のいずれかを含む
[ ] 本文2,000字（±200字）
[ ] 内部リンク2件以上（同カテゴリ or 同エリア）
[ ] 外部リンク：Googleマップ等の信頼性あるリンクを1件

【MEO】
[ ] 最寄駅名・路線名を本文中に自然に記載
[ ] 住所（都道府県・市区町村・エリア名）を本文中に含む
[ ] 周辺施設（カフェ・ホテル・レストラン）を1〜2件、自然な文脈で言及
[ ] TouristAttraction or LocalBusiness の構造化データを実装

【AEO】
[ ] FAQセクション3問以上（基礎知識記事は5問以上）
[ ] FAQ回答は150字以内・施設名＋エリア名＋キーワードを含む直接回答
[ ] H2のうち1つ以上が「〜とは」「〜はなぜ」「〜する方法」形式
[ ] Article構造化データ実装済み
[ ] FAQPage構造化データ実装済み

【品質・世界観】
[ ] 著者：サードプレイスジャパン編集部
[ ] 断言・決めつけ表現がない
[ ] 宣伝・PR的表現がない（カテゴリ10〜14は完全無償記事）
[ ] og:imageに実画像パスを設定（テンプレート変数の未展開がないこと ※緊急修正事項）
[ ] titleタグにテンプレート変数が残っていない（※緊急修正事項）
```
